Release Notes: November 18, 2015

DNX Runner 2.1 RC 1

  • Add support for DNX 1.0 RC 1.
As of RC1, the Visual Studio tooling for class library projects will use the net451 and dotnet5.4 frameworks in your project.json file. Unit tests are a special kind of platform-specific class library, so you must use the dnx451 and dnxcore50 framework names when writing unit tests; otherwise, the xunit.runner.dnx dependency will cause package restore failures.

For more information, please see the Getting Started documentation.
